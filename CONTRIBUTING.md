# Contributing

See [docs/contributing.md](docs/contributing.md) for full contribution guidelines.

## Quick Start

1. Fork the repository
2. Create a feature branch: `git checkout -b improve-godot-debugging`
3. Make your changes
4. Test locally with pi-agent: `pi --skill ./skills/<skill-name> "Test"`
5. Submit a pull request

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).
